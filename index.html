An application that serves as a “sign-up” platform for social dinner meet-ups. Phoenix

Phoenix is a responsive web application that allows users to organize a collection of their personal bucket list goals. We wanted to create a digital and interactive environment where you can keep track of your ideas. This application uses Google OAuth API for user authentication and the Getty Images API to request photos for the activity cards. It is a React.js based app built using Bulma components which implements MongoDB for database purposes. Some other technologies used are Node JS, Express, ES6, and Babel. The future plans for this app include sharing and inviting members to bucket list items, push notifications, implementing Google Flights/Maps, search capabilities, and much more.
